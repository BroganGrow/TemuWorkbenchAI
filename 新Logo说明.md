## 🔧 自定义调整

如果您想调整设计，可以修改以下参数：

### **颜色调整**
编辑 `public/logo.svg`，修改 `fill="#fd7a45"` 为您想要的颜色。

### **透明度调整**
修改 `fill-opacity` 值：
- 当前：`0.9, 0.75, 0.6`（从上到下）
- 可以调整为：`1.0, 0.8, 0.6` 等

### **圆角调整**
修改 `rx` 值：
- 当前：`rx="2"` (小圆角)
- 更圆：`rx="4"` 或 `rx="5"`
- 直角：`rx="0"`

修改后重新运行：
```bash
npm run generate-icons
```
