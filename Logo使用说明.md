## 📝 生成应用图标步骤

### 步骤 1：安装图标生成工具

```bash
npm install sharp --save-dev
```

### 步骤 2：生成图标

```bash
npm run generate-icons
```

这将生成：
- ✅ `icon.png` (1024x1024) - 主图标
- ✅ `icon-16x16.png` ~ `icon-1024x1024.png` - 各种尺寸
- ✅ 打包时自动生成 `.ico` (Windows) 和 `.icns` (macOS)

### 步骤 3：验证图标

生成完成后，重新打包应用即可看到新图标：

```bash
npm run build:win
```

## ❓ 常见问题

### Q: 标题栏 Logo 不显示？
**A:** 确保：
1. `public/logo.svg` 文件存在
2. Vite 开发服务器正在运行
3. 浏览器缓存已清除（Ctrl+F5）

### Q: 应用图标在打包后还是默认图标？
**A:** 请先运行 `npm run generate-icons` 生成 `icon.png`，然后重新打包。

### Q: 如何完全更换为自己的设计？
**A:** 
1. 替换 `public/logo.svg`（标题栏）
2. 替换 `build/icon.png.svg`（应用图标）
3. 运行 `npm run generate-icons`
4. 重新打包
