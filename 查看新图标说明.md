## ❓ 常见问题

### Q: 重启后任务栏图标还是默认的？
**A:** 确保：
1. 已完全关闭所有 Electron 窗口
2. 重新运行 `npm run dev`
3. Windows 可能需要清除图标缓存（重启电脑）

### Q: 打包后图标显示正常吗？
**A:** 是的！electron-builder 会自动处理：
- Windows: 生成 `.ico` 文件
- macOS: 生成 `.icns` 文件
- Linux: 使用 `.png` 文件

### Q: 如何更换图标？
**A:** 替换以下文件后重新生成：
```bash
# 1. 替换 SVG 源文件
build/icon.png.svg

# 2. 重新生成 PNG
npm run generate-icons

# 3. 重启应用
npm run dev
```
